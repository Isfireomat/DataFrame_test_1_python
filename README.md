### Тестовое задание по Pandas
Тестовое задание представлено в файле Middle Python Developer tech task.ipynb

### Запуск
Создание venv: 
```python
python -m venv venv
```
Активировать venv
```shell
source venv/bin/activate
```
Установить poetry:
```
pip install poetry
poetry install
```
Запустить main.py в директории /app
```shell
cd app
python main.py
```
